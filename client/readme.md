# webpack-site-starter 😎
webpack构建普通页面网站，附带tailwindcss

## Feature
✨ 支持TypeScript

✨ webpack-dev-server

✨ 代码压缩、tree-shaking

✨ 模块缓存

✨ 编译进度条

✨ 静态资源处理

✨ tailwindcss

## Usage
推荐使用 `pnpm` !
1. 将此项目克隆到本地

    ```bash
    git clone https://github.com/sunzehui/webpack-site-starter.git
    ```

2. 初始化

   ```bash
   cd webpack-site-starter && pnpm install
   ```
   
3. 启动项目

    ```bash
    pnpm run start
    ```
4. 编译
    
    ```bash
    pnpm run build
    ```
